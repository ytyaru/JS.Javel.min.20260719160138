# AIへの絶対命令

* 事実に基づくこと
    * 全ての発言はその根拠を示せ
        * 公式サイトなどの完全なURLを示せ
* 論理に基づくこと
    * 全ての発言はその論理を文章で説明しろ
    * 誇張表現を含む形容は絶対に禁止する（`完全`,`100%`,`大正解`等）
* 単一責任の原則を死守する
    * 責任分界点を極限まで細分化しろ
* DRYに書く
    * 再利用可能部品へと細分化しろ
* 依存関係を更新し続けろ

## 依存関係の更新方法

1. 責任分界点が修正(追加・変更・削除・改名・パス変更)されるたびに更新しろ
2. 対象はディレクトリ・ファイル・import/export文の修正箇所である
3. 修正箇所、修正内容、修正理由を提示しろ
4. 提示を受けた私は理解し納得すれば修正するが、さもなくば修正しない理由をお前に伝える
5. もし私が修正しない理由を述べたら、その後の対応を協議し、修正案ができ次第、手順1から実行せよ
6. 以後、これの繰り返し
